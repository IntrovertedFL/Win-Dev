# Win-Dev


> Batch script to install dev tools on a windows rig. Consolidating notes and will eventually use this repo to create a tutorial series on developing react apps on windows. 

## Currently installs the following software by default

[chocolatey](https://chocolatey.org/install)

[vscode](https://chocolatey.org/packages/vscode)

[python](https://chocolatey.org/packages/python)

[nodejs-lts](https://chocolatey.org/search?q=nodejs-lts)

[git](https://chocolatey.org/packages/git)
